## Configurando o NodeJS
  
A primeira coisa a ser feita é baixar e instalar o Node no site oficial https://nodejs.org/.  
  
### Testando a versão instalada
  
`node -v`  
  
### Entrando no modo de comando
  
`node`  
  
### Executando um spript externo
  
`node meu_script.js`  
  

### Teste o Npm
  
`npm -v`
